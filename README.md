# SDM-ICD11

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20401111.svg)](https://doi.org/10.5281/zenodo.20401111)

標準病名マスター (MEDIS Standard Disease Name Master、以下 SDM) と WHO ICD-11 (Foundation / MMS) のマッピングデータ。

**Current version:** `v2026.03` (2026 年 3 月初公開) — 変更履歴は [CHANGELOG.md](CHANGELOG.md) を参照。

## 概要

WHO ICD-11 の国内への円滑な導入を推進することを目的に、それをサポートする資料として、ICD10対応標準病名マスター (https://www2.medis.or.jp/stdcd/byomei/index.html) と、WHO ICD-11（https://icd.who.int/en/）の Foundation URI 並びに MMS code との対応づけを東京大学大学院医学系研究科疾患生命工学センター医工情報学部門（今井研究室）が開発し、研究事業の成果物として公開するものです (詳細は [開発経緯と研究助成](#開発経緯と研究助成) 参照)。

## データ

* ICD-10 の全 22 章をカバーする標準病名マスター中の計 **53,329 件** のマッピングエントリを収録しています。

* 標準病名マスターにはカタカナ変換された用語も索引テーブルに多数存在していますが、それは対象から全て省いてあります。

* ICD-10 第 11 章 (消化器系の疾患) における歯科病名は作業の対象外としています。

* ICD-10 の章ごとに CSV ファイルを [`data/`](data/) 配下に配置しています (`chap01.csv` 〜 `chap19.csv`, `chap20-22.csv`)。

* v2026.03 においては標準病名マスターのバージョンは v5.04 をベースとしたものを公開しています。最新版への追随データは追って更新いたします。

### 列仕様

| 列 | 内容 |
| --- | --- |
| ICD10 章 | ICD-10 の章番号 |
| 標準病名 | 標準病名マスター中の病名 |
| 代表・同義 | 標準病名マスターにおける代表語・同義語の区分　`L` = 代表語, `S1` 等 = 同義語 |
| ICD10コード | 対応する ICD-10 コード |
| 交換コード | 標準病名マスターにおける交換コード |
| 関係 (Foundation) | 標準病名と ICD-11 Foundation の粒度関係 (下記参照) |
| ICD11 Foundation URI | 対応する ICD-11 Foundation の URI (Cluster Code 表現) |
| 関係 (MMS) | 標準病名と ICD-11 MMS の粒度関係 |
| MMS | 対応する MMS コード (Cluster Code 表現) |

### 関係記号

| 記号 | 意味 |
| --- | --- |
| `=` | 等価 |
| `<` | 左 (標準病名) の方が粒度が細かい |
| `>` | 右 (ICD-11) の方が粒度が細かい |

### Cluster Code 表現

ICD-11 Foundation URI 列および MMS 列は ICD-11 post-coordination を用いた Cluster Code 表現を用いており、複数のエンティティを組み合わせて 1 つの概念を表現する場合があります。

* `/` 区切り: STEMコード（主要コード）同士の組み合わせ
* `&` 区切り: STEMコードにX章(Extension Code)を修飾コードとして付加した組み合わせ

## 開発経緯と研究助成

このリソースは当初、厚生労働科学研究費補助金 政策科学総合研究事業（政策科学推進研究事業）「社会構造の変化を踏まえた保健医療にかかる施策立案に資する国際疾病分類の国内導入のための研究」（研究代表者：小川俊夫, R2-R4）の分担研究として開発され、その後、厚生労働科学研究費補助金（政策科学総合研究事業（統計情報総合研究事業））「ICD-11の適用を通じて我が国の死因・疾病統計の向上を目指すための研究」（研究代表者：林玲子, R5-R7）の分担研究として WHO ICD-11 のアップデートに対応したメンテナンスが継続されました。リソースの一部の修正については内閣府戦略的イノベーション創造プログラム（SIP）第３期「統合型ヘルスケアシステムの構築」B-2 「統合型の医学概念・知識連結データベースの構築及び医療文書の自動分析基盤の整備」（研究代表者：荒牧英治, R5〜）の助成を受けて継続しています。
